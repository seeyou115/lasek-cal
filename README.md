# 라섹 수술 노모그램 계산기

안과 라섹 수술을 위한 환자 굴절 수치 자동 계산 프로그램입니다.

## 🎯 기능
- 환자의 SPH(구면렌즈 도수)와 CYL(난시 도수) 입력
- 노모그램 규칙에 따른 자동 절삭 수치 계산
- 웹 브라우저에서 바로 사용 가능

## 📦 설치 방법

### 1. Python 설치 (처음 사용하시는 경우)
1. https://www.python.org/downloads/ 접속
2. "Download Python" 버튼 클릭하여 최신 버전 다운로드
3. 설치 시 **"Add Python to PATH"** 체크박스 꼭 선택!

### 2. 프로그램 파일 준비
1. `lasek_calculator.py` 파일을 원하는 폴더에 저장
2. `requirements.txt` 파일도 같은 폴더에 저장

### 3. 필요한 라이브러리 설치
1. 윈도우: 시작 메뉴에서 "cmd" 또는 "명령 프롬프트" 검색 후 실행
2. Mac: "터미널" 실행
3. 저장한 폴더로 이동:
   ```
   cd 폴더경로
   ```
4. 아래 명령어 실행:
   ```
   pip install -r requirements.txt
   ```

## 🚀 실행 방법

1. 명령 프롬프트(또는 터미널)에서 저장한 폴더로 이동
2. 아래 명령어 실행:
   ```
   streamlit run lasek_calculator.py
   ```
3. 자동으로 웹 브라우저가 열리고 프로그램이 실행됩니다!
4. 만약 자동으로 안 열리면: 브라우저에서 `http://localhost:8501` 접속

## 💡 사용 방법

1. SPH(구면렌즈 도수) 입력 (예: -3.50)
2. CYL(난시 도수) 입력 (예: -1.25)
3. "절삭 수치 계산" 버튼 클릭
4. 조정된 절삭 수치 확인

## 🛠️ 코드 수정 방법 (초보자용)

코드를 수정하려면:
1. `lasek_calculator.py` 파일을 메모장이나 Visual Studio Code로 열기
2. 각 단계는 주석으로 설명되어 있습니다
3. 노모그램 규칙을 변경하려면 해당 단계의 조건문 수정
4. 저장 후 프로그램 재실행

예시:
```python
# 이 부분이 SPH가 -2.00일 때의 규칙입니다
elif sph_input == -2.00:
    if -4.0 <= cyl_input <= -0.25:  # CYL 범위 조건
        adjustment = -0.25  # 조정값
        adjusted_sph = sph_input - 0.25  # 최종 SPH
```

## ⚠️ 주의사항
- 본 계산기는 참고용입니다
- 실제 수술 시에는 의사의 전문적인 판단이 최우선입니다
- 계산 결과는 항상 더블 체크하시기 바랍니다

## 📞 문제 해결

### 프로그램이 실행되지 않을 때
1. Python이 제대로 설치되었는지 확인: `python --version`
2. Streamlit이 설치되었는지 확인: `pip list | findstr streamlit`
3. 파일 경로가 맞는지 확인

### 계산 결과가 이상할 때
1. 입력값을 다시 확인
2. 노모그램 규칙이 올바르게 코딩되었는지 확인
3. 필요시 코드의 해당 단계를 찾아 수정
